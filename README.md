# CWGC實用文格式溫習機 v0.1

一個用於練習中文實用文格式的互動式網頁應用程式。

## 功能特色

### 📚 學習內容
- 書信格式
- 建議書格式
- 報告格式
- 專題文章格式
- 演講辭格式

### ✨ 主要功能
- **互動式學習**：點擊選項來選擇文件格式元素
- **即時反饋**：檢查答案並顯示正確/錯誤的視覺回饋
- **進度追蹤**：顯示已完成和未完成的文件類型
- **得分統計**：追蹤正確答題數量
- **清除功能**：一鍵清除所有選擇

### ⌨️ 鍵盤快捷鍵
- **Enter**：確認答案 / 下一題 / 重新開始
- **Escape**：清除所有選擇

### 🎨 用戶體驗改進
- 選中選項帶有縮放效果和陰影
- 橙色「清除選擇」按鈕
- 帶標籤的進度指示器
- 帶表情符號的提示和成功訊息
- ARIA標籤和工具提示以提高可訪問性

## 運行方式

```bash
# 安裝依賴
pip install -r requirements.txt

# 運行應用
python -m flask run
```

然後在瀏覽器中訪問 `http://127.0.0.1:5000`
